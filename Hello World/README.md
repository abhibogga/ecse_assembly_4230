# Hello World in Assembly
## Objective
The objective of this lab is to get the assembly environment set up and start coding in assembly. We will do this by using the code provided in github. The environment to be used is visual studio code. You can follow the hello world assembly pdf if you need extra assistance.
This also introduces SVC calls.  These are O/S independent and provide features/functions for you like writing to the terminal.  See https://developer.arm.com/documentation/den0013/d/Exception-Handling/Other-exception-handlers/SVC-exception-handling

https://stackoverflow.com/questions/49351835/arm-assembly-stdin-stdout-registers-and-programme-os-interaction


## Instructions
1. Copy the HelloWorld.s and makefile into a directory in visual studio code or your favorite IDE
2. Use the terminal and the makefile to compile the .o file use the command "make"
3. Run the program using ./HelloWorld

## Deliverables
1. The code used to display "Hello World from Group XXX" in the terminal with comments.  Note: you will need to change the code in the provide .s file to do this!
2. A screenshot of the terminal output.
3. Describe what the Makefile is and what it is doing for this program.  https://www.gnu.org/software/make/manual/make.html
4. Put all into one pdf files and submit to ELC.
